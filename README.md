## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn
- Git

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/shubhambhatt1251/-Stratify-.git
   cd stratify
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

   Visit `http://localhost:5173` to see your app running!

4. **Build for production**
   ```bash
   npm run build
   # or
   yarn build
   ```
